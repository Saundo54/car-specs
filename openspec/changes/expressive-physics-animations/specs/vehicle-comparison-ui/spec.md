## ADDED Requirements

### Requirement: Comparison Entry Animation
The system SHALL provide a "bubble" animation when a vehicle is added to the comparison list, visually tracing its path to the comparison button.

#### Scenario: Adding vehicle to compare
- **WHEN** the user clicks "Add to Compare" on a vehicle card
- **THEN** a circular "bubble" containing the vehicle image scales up from the click point and animates toward the comparison button in the navigation bar using an expressive spring.
