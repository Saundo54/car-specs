## MODIFIED Requirements

### Requirement: Multi-Select Filtering
The system SHALL provide a filter bottom sheet with options for body type, fuel type, and year range, which animates into view from the bottom.

#### Scenario: Opening the filter sheet
- **WHEN** the user clicks the filter button
- **THEN** the filter sheet slides up from the bottom of the screen with a snappy spring animation.

#### Scenario: Closing the filter sheet
- **WHEN** the user clicks the "Apply" button or the backdrop
- **THEN** the filter sheet slides down and out of view.
