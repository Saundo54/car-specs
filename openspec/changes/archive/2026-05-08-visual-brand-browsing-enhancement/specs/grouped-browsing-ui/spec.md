## MODIFIED Requirements

### Requirement: Brand Selection Flow
The system SHALL transition to a dedicated model/variant list when a specific make is selected, and each vehicle in this list SHALL display its hero image.

#### Scenario: Selecting Toyota
- **WHEN** the user taps on "Toyota" in the make list
- **THEN** the view updates to show all 2018 Toyota models and variants, including a thumbnail or image for each record.
