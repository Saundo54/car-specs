## ADDED Requirements

### Requirement: Frontend Data Connection
The frontend application SHALL load vehicle data from the local `vehicles.json` file instead of using hardcoded mock data.

#### Scenario: Loading vehicles from JSON
- **WHEN** the application initializes
- **THEN** it fetches the `vehicles.json` file and populates the global state (Zustand)

### Requirement: Local Image Rendering
The frontend application SHALL display vehicle images using the local file path generated by the scraper.

#### Scenario: Displaying a local image
- **WHEN** a vehicle detail page is viewed
- **THEN** the hero image is loaded from the local `public/images/vehicles/` directory
