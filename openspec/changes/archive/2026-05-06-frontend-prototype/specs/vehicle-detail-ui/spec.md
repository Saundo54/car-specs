## ADDED Requirements

### Requirement: Categorized Specifications
The system SHALL organize vehicle specifications into distinct, navigable categories: Mechanical, Dimensions, Interior, Exterior, Safety, and Technology.

#### Scenario: Navigating spec categories
- **WHEN** the user selects the "Safety" tab on a vehicle detail page
- **THEN** the system displays the safety-related specifications (ANCAP rating, airbags, AEB, etc.)

### Requirement: Action Bar
The system SHALL provide an action bar on the vehicle detail page for key tasks like "Add to Compare" and "View Source".

#### Scenario: Adding vehicle to comparison
- **WHEN** the user taps the "Add to Compare" button
- **THEN** the vehicle is added to the active comparison shortlist
